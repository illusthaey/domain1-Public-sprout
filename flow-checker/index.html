교육공무직 인사발령 학교별 전입/전출 정리

엑셀(여러 파일 가능)을 업로드하면 학교(부서) 기준으로 모든 직종의 전입(도착) / 전출·퇴직(출발) 명단을 한 화면에 정리합니다.

※ “결원 의심(정원 확인 필요)”는 직종별로 전출·퇴직 > 전입인 경우를 합산한 값입니다.

1) 엑셀 업로드

여러 파일을 한 번에 올릴 수 있습니다. (예: [강릉], [원주] 파일들 동시 선택)

아직 파일이 없습니다.

  • 데이터 행은 기본적으로 연번/순번/번호가 숫자인 행만 사용합니다.
  • 학교명에 줄바꿈/여러 공백이 들어간 경우 학교명 공백/개행 정리 옵션을 켜세요.
  • 파일명이 [강릉]처럼 시작하면 자동으로 “구분(교육지원청)”으로 사용합니다.
고급 설정: 자동 감지 / 컬럼 매핑 / 학교키 규칙

※ 자동 감지는 “성명/번호/직종/부서/비고/임용사항/현직” 패턴으로 잡습니다. 실패하면 아래 수동 매핑을 사용합니다.

※ 여러 교육지원청 파일을 같이 올릴 때는 켜는 것을 권장합니다. (파일명에 [ ] 구분자가 없으면 자동으로 비워둡니다.)

수동 매핑(모든 시트에 동일 적용). 기본(원주형) 매핑: A=연번, B=성명, C=직종, D=전입(부서), F=전출/퇴직(부서), H=비고

2) 학교(부서)별 요약

학교(부서) 기준으로 모든 직종을 합산해 요약합니다. 행 클릭 → 아래 “학교 상세”로 이동합니다.

보기
구분 학교(부서) 전출 전입 결원 의심 순증가 총차이(전출-전입)

※ “결원 의심/순증가”는 직종별 차이를 합산한 값입니다.

3) 전체 결원 의심(학교-직종) 목록

모든 학교의 “직종별 전출>전입”만 모아 빠르게 스캔합니다. 행 클릭 → 해당 학교/직종 상세로 이동합니다.

구분 학교(부서) 직종 전출 전입 결원 의심(전출-전입) 전입없음

4) 학교 상세

학교를 고른 뒤, 직종(전체/개별)을 선택하면 전입·전출 명단을 확인할 수 있습니다.
