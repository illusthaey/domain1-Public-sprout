세외 수납 누락 검출기

농협 입출금 거래내역과 세입세출외현금 수납결의 엑셀을 같은 날짜 기준으로 대조합니다.

  • 판정 핵심은 같은 날짜의 은행 입금합계와 같은 날짜의 수납합계 비교입니다.
  • 농협 거래내역은 실제로 1건 입금이 여러 수납으로 나뉘거나, 반대로 여러 입금이 1건 수납으로 합쳐질 수 있어서 단순 1:1 금액 비교만 하면 오탐이 납니다.
  • 회계연도 시작일(기본 3.1.) 수납 중 최초수납일자가 회계연도 시작일보다 이전인 건은 이월로 보고 자동 제외할 수 있습니다.
  • 서버·DB 없이 브라우저 안에서만 처리합니다. 업로드한 파일은 저장되지 않습니다.

1. 파일 업로드

농협 입출금 거래내역

여기로 농협 엑셀을 드래그 앤 드롭 또는 클릭해서 여러 파일 선택
예: 2025.3~5, 2025.6~8, 2025.9~11, 2025.12~2026.2

세입세출외현금 수납결의

여기로 수납결의 엑셀을 드래그 앤 드롭 또는 클릭해서 파일 선택
한 개만 써도 되고, 필요하면 여러 파일도 합쳐서 검토할 수 있습니다.
파일을 올린 뒤 검토 실행을 누르세요.

2. AI 검토용 프롬프트

날짜 입력값과 이월 제외 옵션을 바꾸면 프롬프트도 같이 바뀝니다.

3. 검토 결과

같은 날짜 기준 합계 비교 + 같은 날짜 내 동일 금액 exact 매칭 참고
파일 0개 날짜 0일

4. 수납 누락 의심 입금

날짜별 비교 상세
상태가 합계일치(묶음/분할)로 보이면, 그 날짜는 총액은 맞지만 입금과 수납이 1:1 금액으로는 바로 안 맞는 날입니다.
묶음/분할 입금 참고
아래 표는 누락이 아니라, 같은 날짜 안에서 입금과 수납이 묶이거나 쪼개진 흔적입니다.
이월 제외 내역
회계연도 개시일 수납 중 최초수납일자가 더 이전인 건을 이월로 보고 비교에서 제외했습니다.
