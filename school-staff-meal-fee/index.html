교직원 급식비 산정 · 에듀파인 업로드 엑셀 생성기

방향키로 셀 이동(엑셀 느낌), 직접 입력 가능 · 엑셀 업로드로 자동 채움 · 결과는 ① 징수내역 집계 + ② 에듀파인 식품비 + ③ 에듀파인 운영비 3개 파일로 다운로드

1) 파일 불러오기 (선택)

템플릿을 안 올려도 됩니다. 이 경우 도구가 학교대상수납요구서식 시트를 새로 만들어 표준 컬럼(선택, *주/야, *계열, *학과, *학년, *반, *번호, *성명, *대상금액)으로 생성합니다.

2) 기본값 입력 (단가·급식일수·파일명)

⤵︎ 10원 단위 절사 규칙 적용

급식일수 칸이 비어 있으면 “기본값 급식일수”를 사용합니다. (엑셀에서 =$D$3 참조하던 느낌을 웹에서 흉내낸 동작)

3) 교직원 명단 편집 (방향키로 이동)

방향키: 이동 · Enter: 아래 · Tab: 오른쪽 · 붙여넣기: Ctrl+V

금액(식품비/운영비/합계)은 자동 계산(회색 셀)입니다. 수정하려면 급식일수 또는 단가를 바꾸세요.

4) 다운로드

모든 계산/변환은 브라우저 내부에서만 수행됩니다(서버 업로드 없음). 단, 이 HTML은 엑셀 처리 라이브러리(SheetJS)를 CDN으로 불러옵니다.
