인사발령 이동 흐름 분석기

인사발령 시행 엑셀을 구조화하여 이동 이력/흐름/집계를 제공하는 참고용 도구입니다. (확정 발령 예측 도구 아님)

0. 라이브러리 상태

내부망에서 외부 CDN 차단 시, vendor 경로에 라이브러리 파일을 배치해야 합니다. (xlsx는 필수 / d3는 선택)
XLSX: 확인 중… D3: 확인 중… Sankey: 확인 중…
로드 순서(자동): ./vendor → /moving/vendor → CDN

1. 자료 업로드

엑셀 파일은 브라우저 내에서만 처리됩니다(외부 전송 없음). 파일 개수 제한은 두지 않습니다.

2. 분석 설정

기관명 정규화

흐름 집계 시 기관명에 괄호/부가 설명이 붙는 경우가 있어, 첫 줄 + 괄호 제거 방식이 실무적으로 안정적입니다.

필터/내보내기

분석 완료 후 필터를 적용하고 CSV로 내보낼 수 있습니다.
