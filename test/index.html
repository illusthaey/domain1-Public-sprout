2026학년도 연차유급휴가 부여일수 계산기

(기준일: 2026-03-01 · 토요일 제외 · 1일 소정근로시간 반영)

1. 파일 업로드

인사기록(근속/직종/근무형태)

· 엑셀: 연차 관련 - 인사기록-교육공무직.xlsx
· 주요: 성명/개인번호/최초임용일/근무형태(상시/비상시)/직군(교육공무직/특수운영) 등

근무상황목록(복무 내역)

· 엑셀: 연차 관련 - 2025회계 근무상황목록 교육공무직 전체 (문막초).xlsx
· 주요: 시작/종료, 종별, 사유, 결재상태 등

데이터 미리보기

인사기록 로드
근무상황 로드

2. 학사일정 입력(여름/겨울방학)

· 비상시 직종(방학중 비상시)의 학기중/연간 출근율 계산에 사용됩니다.
· 입력 후 학사일정 적용 버튼을 눌러주세요.

3. 복무 자동분류 + 수동보정

· 업로드한 근무상황 목록을 자동으로 산정제외 / 결근성 / 출근간주 / 검토로 분류합니다.
· 자동분류가 애매하면 검토로 두고, 아래에서 최종분류를 수동으로 바꿀 수 있습니다.

성명 개인번호 기간 종별 사유 결재상태 자동분류 최종분류 방학중근무가산 시간/일수(원본) 파일

수동 복무내역 추가(선택)

· 엑셀에 없는 누락분이나, 방학중 실제 근무(우선채용 등)를 직접 입력할 때 사용합니다.

비상시 직종: 방학중 근무(출근 간주) 날짜 추가 입력

·방학 기간에 실제 근무한 날짜를 입력하면, 해당 일수만큼 연간 출근율(분자)에 가산합니다. (학사일정이 적용되어 있어야 방학 일수로 인식됩니다)
※ 출근율가산(3/4일) 판정은 학기중 출근율 80% 기준으로 적용되며, 연간 출근율은 참고용 표시입니다.

기간 메모 방학일수(토요일 제외) 삭제

4. 연차 유급휴가 부여일수 계산 실행

성명 개인번호 직군 근무형태 근속(년) 출근율(학기) 출근율(연간) 출근율(비례) 기본 출근율가산(학기중80%) 근속가산 표준합계 최종부여 판정
· 자동분류/출근율 계산은 학교 복무기록의 표준적 패턴을 기준으로 하며, 기관별 세부 운영에 따라 수동 보정이 필요할 수 있습니다.
