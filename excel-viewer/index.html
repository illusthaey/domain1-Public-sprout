엑셀 뷰어

엑셀 파일을 드래그해서 올리고 반영하기를 누르면, 선택한 시트를 웹 표로 보여줍니다. 서버 업로드 없이 브라우저 안에서만 처리됩니다.

클릭해서 파일 선택도 가능합니다.

파일을 올리면 여기에서 상태를 안내합니다.

동작 특징

  • 병합 셀, 열 너비, 행 높이, 빈칸 위치를 최대한 유지합니다.
  • 셀에 링크가 있으면 클릭 가능한 링크로 표시합니다.
  • 큰 시트는 브라우저가 잠깐 버벅일 수 있어, 반영 전에 한 번 더 확인합니다.
  • 웹에서 완전히 같은 서식까지 100% 복제하는 건 한계가 있어, 핵심은 내용과 구조 재현에 맞췄습니다.

운영 배포 시에는 SheetJS 스크립트를 CDN 대신 로컬 파일로 내려받아 두는 편이 더 안정적입니다.
