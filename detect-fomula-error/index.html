엑셀 수식 깨짐 탐지기 (완전 오프라인 · 공통 CSS 충돌 회피판)

브라우저 안에서 ZIP/XML을 직접 읽어, 주변 수식 패턴과 다른 셀·수식이 빠진 셀·오류 값을 가진 셀을 찾아줍니다. 공통 style.css / footer.js가 함께 로드되는 환경에서도 버튼 글씨, 카드, 칩 필터가 깨지지 않도록 앱 전용 클래스로 분리했습니다.

파일을 끌어놓거나 선택하세요
이 완전 오프라인판은 .xlsx / .xlsm 을 지원합니다.
선택된 파일 없음
좌우/상하 몇 칸까지 같은 패턴을 볼지 정합니다.
검사 방향 / 옵션
이 도구는 계산값을 다시 계산해서 맞다/틀리다를 판정하는 방식이 아니라, 셀 기준 상대참조 패턴을 정규화해서 주변과 다른 셀을 찾는 방식입니다. 그래서 복사형 수식에는 강하지만, 의도적으로 다른 합계행·누계·특수 예외행은 “의심”으로 잡힐 수 있습니다.
완전 오프라인을 위해 외부 라이브러리를 넣지 않은 버전이라서, 지원 범위는 .xlsx / .xlsm 위주입니다. .xls / .xlsb / .ods까지 모두 먹이려면 SheetJS 같은 파서를 통째로 내장한 더 큰 파일이 필요합니다.
공통 CSS와의 충돌을 피하려고 .btn, .card, .chip, .row, .grid 같은 범용 이름을 쓰지 않고 앱 전용 ffc-* 클래스를 사용합니다.
브라우저에서 처리하므로 원본 파일은 서버로 전송되지 않습니다. 아주 큰 파일은 시간이 조금 걸릴 수 있습니다. 최신 Chrome/Edge 계열 브라우저에서 가장 안정적으로 동작합니다.
